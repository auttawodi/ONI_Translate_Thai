# ONI_Translate_Thai
โปรเจคแปลไทย เวอร์ชั่น 0.1

การแปลนี้จะช่วยให้ง่ายต่อการเล่น ทั้งในด้านภาษาและเกมเพลต่างๆ ทำให้เข้าใจมากขึ้น
แต่ไม่สามารถแปลตัวคนเดียวหมดได้ หากใครต้องการช่วยแปลสามารถช่วยได้

วิธีการแปล
ดาวน์โหลดโปรแกรมแปลที่ https://poedit.net/
ดาวน์โหลดไฟล์แปลไทยที่ https://github.com/auttawodi/ONI_Translate_Thai คลิกที่ Clone or download และคลิก Download ZIP

1.เปิดโปรแกรม Poedit ขึ้นมา 
2.คลิกที่ Open เลือกไฟล์ strings.po ที่ดาวน์โหลดมา (ควรแยกไฟล์พร้อมกับ strings.mo)
3.คลิกที่ Source text เลือกบรรทัดที่ยังไม่ถูกแปล เติมคำแปลได้ในช่อง Translation 

!ข้อควรระวัง!
อย่าแปลคำในเครื่องหมาย {} กับ <> เพราะคำพวกนี้ต้องใช้ในเกม
